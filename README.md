Прога созданная мной (aka нейрохрючево) для зубрежки различного рода тестов.

Принимает оно JSON файл с содержанием теста, тесты это ключи в которых уже лежат объекты, всегда вариантов ответов должно быть 4 иначе прога не работает, всегда скармливаю ворд документы в дипсик обяъсняя шаблон теста и он выдает готовый json файл 
который уже удобоварим для проги.
